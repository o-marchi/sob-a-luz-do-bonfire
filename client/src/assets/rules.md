## Regras do sistema de votos:

<br>

- Nós temos uma nova moeda no grupo: Token
- Você vota em jogos usando Tokens
- Todo o começo de ciclo você ganha um Token
- Se você jogar o jogo do mês, você ganha um Token
- Se você terminar o jogo, você ganha mais um Token
- Se você participar do encontro, você ganha um Token, tendo jogado o jogo ou não
- Ao sugerir um jogo, você gasta um Token
- Ao votar num jogo, você vota com todos os seus Tokens restantes
- Os Tokens não são cumulativos e resetam a cada ciclo.

<br>

## Regras para recomendar jogos

Nós não temos regras específicas para recomendar jogos.
Porém nós recomendamos jogos curtos, no máximo 15 horas (main + extras) segundo o [How Long To Beat](https://howlongtobeat.com/), mas preferencialmente de 5 a 10 horas.
